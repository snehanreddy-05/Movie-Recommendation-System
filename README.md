# Movie Recommendation System

A Netflix-style movie recommender built with Python and Streamlit.

## Features

- Movie Recommendations
- TMDB Posters
- Ratings
- Release Year
- Movie Overview
- Trending Movies

## Technologies Used

- Python
- Streamlit
- Pandas
- Scikit-Learn
- TMDB API

## How to Run

pip install -r requirements.txt

streamlit run app.py
